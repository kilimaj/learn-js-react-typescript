Promises

  • A promise is an object that represents the eventual completion or failure of an asynchronous operation.
  • An asynchronous operation is a type of operation that doesn't block the program's execution and allows other code to run while it's being processed.
  • Promises can be used to handle the results of asynchronous operation, such as network request or a file read, that take some time to complete

Promises 3 states

  • Pending - starts in a pending state
  • Fulfilled - if the asynchronous operation succeeds
  • Rejected - if the asynchronous operation fails

Promises provide a way to handle asynchronous results in a manageable and predictable manner.

Async Await keywords

  • Async keyword is used to create asynchronous functions in JS
  • When you declare a function with async keyword you can use the await keyword inside the function to wait for the results of an asynchronous operation
